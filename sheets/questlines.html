Lightseeker Hyetta

  • Finishing Hyetta's questline requires inheriting the frenzied flame. If you don't want the Lord of Frenzied Flame ending, simply beat the game with another ending first (without starting NG+) before inheriting the flame. If you regret inheriting it, the frenzied flame can be tamed by using Miquella's Needle at Dragonlord Placidusax's arena in Farum Azula.
